Tạo Project Flutter bằng VS Code

Posted: November 18, 2019 by anhkevin

VS Code là một trình biên tập mã tương đối nhẹ và hỗ trợ tốt, đặc biệt nó miễn phí. Trong VS Code có extension hỗ trợ Flutter.

Bước 1: Trên VS Code cài extension để hỗ trợ Flutter

Bước 2: Khởi động lại VS code và tạo Project

  • Nhấn Ctrl-Shift-P trên Windows.
  • Nhập vào Flutter: New Project
  • Nhập tên Project và nhấn Enter

=> Đã tạo xong Project

  • Build và chạy dự án bằng cách nhấn F5 hoặc chọn Debug> Start Debugging

Một số lỗi thường gặp khi build

Lỗi 1:

Nếu gặp lỗi tương tự phía trên thì xóa các thư mục không có file tên là llvm-rs-cc  ở trong C:\Users\xxx\AppData\Local\Android\Sdk\build-tools và build lại

Lỗi 2:Do bản Android Q beta không chạy được. Nên hạ phiên bản Android xuống. Có thể tạo Virtual Device mới với phiên bản Android thấp hơn.
